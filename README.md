# change_calculator
Calculates the change for a certain transaction (Canadian Money)


## How this calculator works

The user has to put in the total cost, and the amount that they give

For example, if the cost was 55.30$ and the person gave 60$

Output: 

Total Change Amount: 4.70$
Give: 2 Toonies, 2 Quarters, 2 Dimes


Also checks if the person gives below the change, if so then tell them give more money.
